WebAssembly

"What is WebAssembly?"

WebAssembly is a programming language made for websites.
The only language supported back in the old days of 2016 for HTML interactivity was JS
(JavaScript).

In March of 2017, WebAssembly was released to the public as a gateway between JavaScript, HTML, and other languages such as C++,
Mainline Java, Python, or other languages. According to source one, "WASM was never created as a replacement for Javascript, and it can't because the language is so low-level.
Instead, it's specifically made to complement Javascript and addresses its performance issue when conducting computation and memory-intensive tasks"(ymc9, What are people building with webassembly? 2023)
There is an API used for JavaScript integration known as WebAssembly JavaScript Interface, used specifically to run JS code faster and more efficient than what barebones HTML integration could likely provide. This uses the WASM code to slash the load time in half if not more.
